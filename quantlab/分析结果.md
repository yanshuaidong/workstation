## 📈 标签分布（预测目标）

## 🎯 策略核心结论

1. **低胜率 + 高盈亏比** = 盈利模式
   - 虽然只有 42% 的交易盈利，但盈利时赚得多（平均 +3%），亏损时亏得少（平均 -2%）

2. **风险控制优秀**
   - 最大回撤仅 9.47%，夏普比率高达 2.15，说明收益稳定

3. **多空均衡**
   - 多头空头交易数量和胜率接近，策略在涨跌行情中都能获利

4. **移动止损是主要退出方式**
   - 占比约 50%，说明策略善于"让利润奔跑"同时保护本金

## 📚 专业术语详细解释

### 模型相关

| 术语 | 解释 |
|------|------|
| **AUC (Area Under Curve)** | 模型区分能力的指标，范围 0-1。0.5 表示随机猜测，0.7 以上表示有较好区分能力，0.9 以上表示非常优秀。本策略测试集 AUC 约 0.72-0.73，说明模型能较好地区分"会涨/跌"和"不会涨/跌"的情况。 |
| **LightGBM** | 一种高效的机器学习算法，属于"梯度提升树"家族。可以理解为让很多"弱智小决策"组合成一个"聪明大决策"。特别擅长处理表格数据，训练速度快。 |
| **特征 (Feature)** | 用于预测的输入变量。比如"过去5天涨幅"、"成交量变化"等。模型根据这些特征来判断未来走势。本策略使用了53个特征。 |
| **标签 (Label)** | 我们想要预测的目标。本策略的标签是"未来5天最大涨幅是否超过3%"（多头）和"未来5天最大跌幅是否超过3%"（空头）。 |
| **阈值 (Threshold)** | 触发信号的临界点。模型输出的是0-1之间的概率，只有概率超过阈值时才触发交易。比如多头阈值0.6296意味着只有概率大于62.96%才做多。 |
| **验证集/测试集** | 验证集用于调参选模型（2023年数据），测试集用于最终评估真实表现（2024年至今）。分开是为了避免"开卷考试"——用没见过的数据检验模型。 |
| **Precision (精确率)** | 在所有"模型说会涨"的预测中，实际涨的比例。精确率高说明模型"说涨就涨"，虚假信号少。 |
| **Recall (召回率)** | 在所有"实际会涨"的样本中，模型正确预测出来的比例。召回率高说明模型不会错过太多机会。 |
| **F1 Score** | 精确率和召回率的综合指标，是两者的调和平均值。用于在精确率和召回率之间找平衡。 |
| **scale_pos_weight** | 正负样本不平衡时的补偿权重。如果正样本（涨）少，就给正样本更大权重，让模型更关注正样本。 |

### 回测相关

| 术语 | 解释 |
|------|------|
| **胜率 (Win Rate)** | 盈利交易占总交易的比例。42%意味着每100笔交易约有42笔赚钱。胜率不是越高越好，关键看盈亏比。 |
| **盈亏比 (Profit Factor)** | 平均盈利金额 ÷ 平均亏损金额。1.81意味着赚钱时平均赚1.81元，亏钱时平均只亏1元。大于1就能盈利。 |
| **累计收益** | 策略从开始到结束的总收益率。101.28%意味着初始资金翻了一倍多。 |
| **年化收益** | 把收益换算成"如果持续一整年"的收益率。方便不同时间长度的策略进行比较。45.9%年化收益在量化策略中属于优秀水平。 |
| **最大回撤 (Max Drawdown)** | 从最高点到最低点的最大跌幅。9.47%意味着在最糟糕的时期，账户从峰值最多跌了9.47%。回撤越小，策略越稳定。 |
| **夏普比率 (Sharpe Ratio)** | (收益率 - 无风险利率) ÷ 收益波动率。衡量"每承担一单位风险能获得多少收益"。2.15表示每承担1%的风险波动，能获得2.15%的超额收益。一般大于1算不错，大于2算优秀。 |
| **止损 (Stop Loss)** | 亏损达到预设比例（2%）时强制平仓，防止亏损扩大。"割肉"保命。 |
| **止盈 (Take Profit)** | 盈利达到预设比例（5%）时主动平仓，锁定利润。"落袋为安"。 |
| **移动止损 (Trailing Stop)** | 随着价格上涨，止损线也跟着上移。比如从最高点回落1.5%就平仓。既能保护利润，又能让利润继续增长。 |
| **持仓天数** | 每笔交易从开仓到平仓的时间。平均2.56天说明这是短线策略。 |

### 数据相关

| 术语 | 解释 |
|------|------|
| **预热期 (Warmup Period)** | 计算某些指标（如60日均线）需要足够的历史数据。前60天数据只用于计算指标，不参与训练。 |
| **持仓量 (Open Interest)** | 市场上未平仓合约的总数。持仓量增加说明有新资金进场，减少说明资金离场。 |
| **ATR (Average True Range)** | 平均真实波幅，衡量价格波动剧烈程度的指标。ATR大说明行情波动大，适合趋势策略。 |
| **RSI (Relative Strength Index)** | 相对强弱指标，衡量近期涨跌力度。0-30超卖（可能反弹），70-100超买（可能回调）。 |
| **布林带 (Bollinger Bands)** | 以均线为中轨，上下各加减2倍标准差形成的通道。价格突破上轨可能是强势信号，跌破下轨可能是弱势信号。 |
