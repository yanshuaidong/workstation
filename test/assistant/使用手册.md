# Assistant 辅助决策使用手册

## 1. 功能说明

当前 assistant 模块已经接入：

- 从 `fut_strength` / `fut_daily_close` 读取基础行情
- 计算 7 个资金信号并写入 `assistant_signals`
- 生成 4 条机械策略建议并写入 `assistant_operations`
- 可选执行机械账户开平仓，更新 `assistant_positions` / `assistant_account_daily`
- 生成本地 LLM 提示词文件 `logs/llm/YYYY-MM-DD-prompt.md`
- 提供 `write_llm_decision.py` 作为 LLM 写库 CLI
- 提供后端 `/api/assistant/*` 查询接口
- 提供前端“辅助决策”页面

---

## 2. 数据前置

assistant 依赖 `fut_pulse` 先把两张基础表准备好：

- `fut_strength`
- `fut_daily_close`

日常推荐顺序：

```bash
cd /Users/zxxk/ysd/ysdproject/workstation/database/fut_pulse
python main.py today
```

如果历史 close 还没补齐，再执行：

```bash
python main.py close-history --start 2026-04-01 --end 2026-04-17
```

注意：

- `assistant` 取最新交易日时直接依赖 `fut_daily_close`
- 如果 `fut_daily_close` 缺失或没补到最新日期，`daily_run.py` 会拿不到正确的行情日期

创建 assistant 自己的四张业务表：

## 4. 每日运行

只生成信号、建议和 LLM 提示词，不执行机械账户：

```bash
cd /Users/zxxk/ysd/ysdproject/workstation/test/assistant
python src/daily_run.py --signal-date 2026-04-17
```

执行机械账户开平仓：

```bash
python src/daily_run.py --signal-date 2026-04-16 --execute-mechanical
```

回放一个时间区间：

```bash
python src/daily_run.py --start-date 2026-04-01 --end-date 2026-04-17
```

强制重放机械账户仅在你明确知道当前账户状态时使用：

```bash
python src/daily_run.py --signal-date 2026-04-17 --execute-mechanical --force-replay
```

---

## 5. 推荐运行顺序

推荐把数据链路和 assistant 链路按下面顺序执行：

1. `database/fut_pulse/main.py today`
2. 如需补历史 close，再执行 `database/fut_pulse/main.py close-history ...`
3. `test/assistant/src/daily_run.py`

如果当天采集曾经上传失败，恢复顺序建议是：

1. `database/fut_pulse/main.py upload-only`
2. `python src/daily_run.py --signal-date ...`

补充说明：

- `upload-only` 如果推断为今日模式，会自动补今日 `fut_daily_close`
- `upload-only` 如果推断为历史模式，只会补 `fut_strength`，历史 close 仍然要走 `close-history`

## 6. LLM 决策流程

`daily_run.py` 运行后会生成：

```text
test/assistant/logs/llm/YYYY-MM-DD-prompt.md
```

本地 AI 工具读取该文件后，通过 CLI 写入决策。

开多：

```bash
cd /Users/zxxk/ysd/ysdproject/workstation/test/assistant
python src/write_llm_decision.py \
  --signal-date 2026-04-17 \
  --action open_long \
  --variety_id 28 \
  --variety_name PTA \
  --size_pct 0.3 \
  --strategy 1A \
  --reasoning "主力连续走强且散户反向，决定跟随做多。"
```

平仓：

```bash
python src/write_llm_decision.py \
  --signal-date 2026-04-17 \
  --action close \
  --variety_id 28 \
  --variety_name PTA \
  --reasoning "趋势减弱，先落袋为安。"
```

仅记录 hold：

```bash
python src/write_llm_decision.py \
  --signal-date 2026-04-17 \
  --action hold \
  --variety_id 28 \
  --variety_name PTA \
  --reasoning "信号未破坏，继续观察。"
```

所有 LLM 调用都会追加归档到：

```text
test/assistant/logs/llm/YYYY-MM-DD-calls.jsonl
```
