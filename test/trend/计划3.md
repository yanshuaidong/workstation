# 计划3：从关系验证转向条件化交易因子筛选

> 基于计划1和计划2的结论，提出更理性的研究方向：**不再继续寻找“对全部品种都成立”的统一资金信号，而是先识别品种属于哪一类资金行为，再在各自适配的场景下筛选可交易因子。**

---

## 一、为什么需要计划3

前两次实验已经给出了比较清晰的边界：

1. **计划1说明：单日资金值太弱**
   - `main_force`、`retail` 与日收益率整体只有弱相关，平均相关系数仅在 `±0.11` 左右
   - 因此，**直接拿单日值做交易信号不可靠**

2. **计划2说明：趋势比单点更有信息，但纯趋势方法仍有局限**
   - `main_force` 连续趋势段与价格方向的一致率达到 **69.2%**
   - 但真正强信号集中在 **5 天及以上** 的持续趋势，说明短趋势大多是噪声
   - 趋势确认需要时间，**纯趋势跟随存在交易滞后**

3. **两次实验共同说明：品种之间差异很大**
   - **能源化工、黑色系** 更接近“趋势共振型”
   - **部分有色金属** 更接近“资金领先型”或“逆势建仓型”
   - 这意味着：**统一策略不合理，按品种风格分型更合理**

4. **底部/顶部反转假设当前不值得继续主攻**
   - 计划2中 H2/H3 没有得到支持
   - 在现有样本量下，直接研究“触底必反弹”容易陷入小样本噪声
   - 更理性的方向是：**寻找条件更明确、定义更稳定、可分层检验的交易因子**

---

## 二、计划3的核心判断

### 判断1：资金信号的有效信息不在“绝对值”，而在“结构关系”

真正有交易价值的，不是某一天 `main_force=20` 还是 `-20`，而是以下结构是否同时出现：

- **持续性**：主力行为是否连续数日同向
- **相对性**：主力与散户是否背离
- **未兑现性**：资金是否已经先动，但价格还没完全反映
- **风格适配**：该品种到底属于趋势共振，还是逆势吸筹

### 判断2：计划3不再优先回答“有没有关系”，而是回答“能不能交易”

因此，研究指标将从“相关系数、方向一致率”进一步升级到：

- 因子对未来 `1/3/5` 日收益的 **IC / RankIC**
- 分层组合的 **单调性**
- 加入手续费/滑点后的 **可交易性**
- 因子是否只在少数品种有效，还是在某一类风格中稳定有效

### 判断3：计划3的第一目标不是做出复杂模型，而是筛出少数值得继续跟踪的因子

计划3不追求一步到位预测所有价格变化，而是优先完成三件事：

1. 给 55 个品种做一个**资金行为分型**
2. 构造一组**简单但可解释**的候选因子
3. 找出其中**真正有交易价值**的 2~3 个方向

---

## 三、核心假设（更理性的猜想）

### 假设 H1：品种存在稳定的资金行为风格

不是所有品种都应该用同一套资金信号。更合理的猜想是，至少存在以下 4 类：

| 风格 | 特征 | 对应现象 |
|------|------|---------|
| **趋势共振型** | 主力趋势与价格趋势大体同向 | 计划2中的黑色系、能源化工 |
| **资金领先型** | 主力变化领先价格 1~3 天 | 计划1中的沪铝、沪铜、菜粕等 |
| **逆势吸筹型** | 主力与价格同期可能负相关 | 计划1中的原木、碳酸锂、沪镍等 |
| **噪声型** | 无明显稳定规律 | 不应纳入重点交易池 |

### 假设 H2：真正可交易的 alpha 来自“持续性 + 背离 + 未兑现”的组合

单一因子很难稳定，但若满足以下结构，信号质量会明显提升：

- 主力连续增强
- 散户方向相反或明显落后
- 价格尚未完全跟随

这类信号比“单日强弱值”或“是否触底”更接近可交易 setup。

### 假设 H3：趋势因子适用于趋势共振型品种，逆势因子适用于逆势吸筹型品种

也就是说，**因子本身没有绝对好坏，只有是否适配该品种的资金行为模式**。

### 假设 H4：加入价格确认后，资金因子的可交易性会优于单独使用资金因子

纯资金因子容易过早，纯价格信号又容易过晚。更合理的猜想是：

- 资金因子负责提供方向和先验
- 价格确认负责过滤噪声和极端反向样本

## 四、计划3拟重点研究的交易因子

### 4.1 因子池设计原则

计划3只研究少数几个可解释、可落地的候选因子，不再做大规模“堆指标”。

所有因子都遵循以下原则：

- 先在**品种内标准化**，避免跨品种口径不一致
- 优先使用 `5` 日窗口，因为计划2已证明 `5 天+` 更接近有效趋势
- 因子既要能做时间序列判断，也要能做横向排序

### 4.2 候选因子清单

| 因子名 | 核心定义 | 经济含义 | 预期适用品种 |
|------|---------|---------|-------------|
| **F1 资金持续性因子** | `main_force` 近 5 日累计变化 + 连续同向天数 | 主力是否在持续推动同一方向 | 趋势共振型 |
| **F2 主散背离因子** | `main_force` 变化减去 `retail` 变化 | 主力与散户是否出现“剪刀差” | 趋势共振型、资金领先型 |
| **F3 资金领先未兑现因子** | 资金变化强，但价格 5 日涨跌幅仍弱 | 主力先动、价格未充分反映 | 资金领先型、逆势吸筹型 |
| **F4 价格确认因子** | 资金方向与价格突破/均线方向一致 | 用价格行为确认资金信号 | 全部有效风格中的过滤器 |
| **F5 逆势吸筹因子** | 价格走弱但主力改善，且主散差收敛/转正 | 主力在下跌中吸筹，而价格尚未转向 | 逆势吸筹型 |

### 4.3 因子公式建议

为避免一开始过度复杂，先用下面这版定义作为研究起点：

#### F1：资金持续性因子

```text
F1 = zscore( main_force[t] - main_force[t-5] ) * (同向连续天数 / 5)
```

解释：
- 衡量主力近 5 日是否在持续增强或持续减弱
- 连续天数越长，说明信号越不是随机扰动

#### F2：主散背离因子

```text
F2 = zscore( (main_force[t] - main_force[t-5]) - (retail[t] - retail[t-5]) )
```

解释：
- 主力与散户方向越相反、幅度差越大，因子值越高
- 这是对计划2中“散户反向指标”结论的直接利用

#### F3：资金领先未兑现因子

```text
F3 = zscore(main_force_5d_change) - zscore(price_return_5d)
```

解释：
- 如果资金已经明显转强，而价格 5 日表现仍弱，说明存在“未兑现空间”
- 这是计划1中领先-滞后结论的自然延伸

#### F4：价格确认因子

```text
F4 = 1( close > MA5 且 close 创近3日新高 ) * sign(F1 + F2)
```

做空方向对称处理。

解释：
- F4 不是独立 alpha，而是过滤器
- 它的作用是把“看起来有资金信号，但价格没有跟”的样本分成“值得等”和“已经失效”两类

#### F5：逆势吸筹因子

```text
F5 = 1(price_return_5d < 0) * zscore(force_spread_5d_change)
```

其中：

```text
force_spread_5d_change = (main_force - retail)[t] - (main_force - retail)[t-5]
```

解释：
- 当价格仍在下行，但主散差开始明显向主力一侧改善时，可能代表逆势吸筹
- 仅在“逆势吸筹型”品种中评估，不做全市场泛用

## 五、计划3与前两次实验的关键区别

| 维度 | 计划1 | 计划2 | 计划3 |
|------|------|------|------|
| 核心问题 | 单日资金值是否与价格相关 | 资金趋势是否与价格共振 | **哪些条件化因子可以用于交易** |
| 分析对象 | 单日值 | 连续趋势段 | **日频因子 + 品种风格标签** |
| 假设风格 | 全品种统一关系 | 全品种统一趋势关系 | **按风格分型后再测试因子** |
| 评估方式 | 相关系数 / Granger | 一致率 / 条件概率 / 回测 | **IC / 分层收益 / 成本后回测** |
| 输出目标 | 证明有无关系 | 找趋势型策略 | **筛出可交易因子与可交易品种池** |

## 六、研究阶段设计

### 阶段一：建立因子研究样本

**目标**：把现有数据整理成可用于因子检验的 panel 数据集。

#### 具体工作

1. 继续将 `fut_strength` 与 `fut_daily_close` 按 `(variety_id, trade_date)` 合并
2. 计算未来收益标签：
   - `fwd_ret_1d`
   - `fwd_ret_3d`
   - `fwd_ret_5d`
3. 计算基础特征：
   - `main_force_1d_change`
   - `main_force_5d_change`
   - `retail_5d_change`
   - `force_spread`
   - `force_spread_5d_change`
   - `price_return_3d / 5d`
   - `rolling_vol_5d`
4. 对各品种做滚动标准化（z-score 或百分位），保证横向可比

### 阶段二：给品种做风格分型

**目标**：先判断一个品种属于哪类资金行为，再决定用什么因子。

#### 分型依据

每个品种计算以下统计量：

- `同期相关性`：计划1中的 `main_force vs return`
- `领先相关性`：`main_force_delta(t)` 对 `return(t+1~t+3)` 的最大相关
- `趋势共振率`：计划2中的方向一致率
- `5天+趋势占比`：识别出的强趋势占全部趋势的比例

#### 初步分型规则

| 类型 | 判定思路 |
|------|---------|
| **趋势共振型** | 趋势一致率高，且 5 天+ 趋势占比较高 |
| **资金领先型** | 领先相关性显著，但同期相关性未必高 |
| **逆势吸筹型** | 同期相关性为负，或趋势共振率偏低但领先关系存在 |
| **噪声型** | 各项指标均不稳定或样本过少 |

如果单个品种样本太少，则退一步先按**板块分型**，再在板块内挑选代表品种。

### 阶段三：候选因子检验

**目标**：筛选出真正有统计意义且有交易意义的因子。

#### 检验方法

1. **IC / RankIC**
   - 分别计算 F1~F5 对未来 `1/3/5` 日收益的 IC
   - 同时看全样本、按风格分组、按板块分组结果

2. **分层收益检验**
   - 每日把品种按因子值分成 5 组
   - 检查高分组到低分组的收益是否单调
   - 重点看 `Top - Bottom` 组合收益

3. **命中率检验**
   - 因子方向与未来收益方向一致的比例
   - 对于低频信号，记录触发次数，避免“高胜率但几乎不触发”

4. **稳健性筛选**
   - 窗口从 `3/5/7` 日做敏感性分析
   - 不允许一个因子的结论完全依赖单一极端品种

### 阶段四：构建风格化组合信号

**目标**：不是直接挑最强单因子，而是做“风格内最优组合”。

#### 组合思路

| 风格 | 优先因子 | 组合思路 |
|------|---------|---------|
| **趋势共振型** | F1 + F2 + F4 | 主力持续性 + 主散背离 + 价格确认 |
| **资金领先型** | F3 + F2 | 资金先行 + 主散差强化 |
| **逆势吸筹型** | F5 + F3 | 吸筹迹象 + 未兑现空间 |
| **噪声型** | 无 | 暂不纳入交易池 |

#### 信号形式

- 时间序列信号：单品种做多/做空评分
- 横截面信号：同一天在有效品种池内做排序，选择最强多头和最强空头

### 阶段五：交易回测与实用性评估

**目标**：从“统计有效”进一步走到“可以交易”。

#### 回测设计

1. **持有周期**
   - `1 日`
   - `3 日`
   - `5 日`

2. **交易方式**
   - 只做多
   - 多空对冲
   - 分风格独立回测

3. **过滤条件**
   - 仅交易风格分类明确的品种
   - 仅在因子值超过阈值时触发
   - 可加入波动率过滤，避免极端噪声日

4. **评估指标**
   - 胜率
   - 平均收益
   - 盈亏比
   - 最大回撤
   - 换手率
   - 扣除手续费和滑点后的净收益

### 阶段六：输出最终研究结论

计划3最终要回答的不是“资金有没有用”，而是下面 4 个更务实的问题：

1. 哪些品种可以直接剔除，不值得继续跟踪？
2. 哪些品种适合趋势型因子，哪些适合逆势型因子？
3. F1~F5 中，哪些因子在交易成本后仍保留 alpha？
4. 是否能形成一个简单、低频、可解释的交易框架？

## 七、计划3的判定标准

为了避免继续陷入“看上去有点效果”的模糊结论，计划3设置明确通过标准。

### 因子可继续跟踪的最低标准

一个候选因子至少需要满足以下条件中的大部分：

1. 在某一风格或某一板块中，`IC` 为正且方向稳定
2. 分层收益存在基本单调性，高分组显著优于低分组
3. 扣除手续费/滑点后，组合收益仍为正
4. 结果不是由单一极端品种驱动
5. 触发频率足够，不是“只出现 1~2 次”的伪信号

### 应当放弃的情形

若出现以下情况，则直接判定该因子暂不值得继续投入：

- 只在全样本有效，但一分板块/风格就失效
- 对参数极度敏感，窗口一改就翻转
- 高收益完全来自单个极端品种
- 扣除成本后失去全部优势

## 九、计划3的现实约束

1. **样本仍然偏短**
   - 当前时间跨度仍不足以支持复杂模型
   - 所以计划3必须保持低参数、低自由度、强解释性

2. **分型优先于拟合**
   - 若样本不够，不应强行细分到单品种最优参数
   - 先做板块级/风格级规律，再下沉到品种

3. **因子优先于策略花样**
   - 计划3的重点不是设计很多买卖规则
   - 而是确认哪些因子本身值得保留，哪些该尽快淘汰

4. **交易成本必须前置考虑**
   - 如果信号必须高频切换才能有效，那么它大概率不适合真实交易
   - 因此计划3会优先看 `3~5` 日持有期的低频因子

## 十、结论性方向判断

如果把前两次实验的结果合在一起，一个更理性的下一步不是继续问：

- “`main_force` 到底是不是预测价格？”

而是改问：

- “**在哪类品种里，`main_force` 的哪种结构，能转成成本后仍有效的交易因子？**”

因此，计划3的最佳方向是：

1. **先分型，再研究因子**
2. **优先研究 F1/F2/F3 这三个最有逻辑基础的因子**
3. **把 F4 作为过滤器，而不是主因子**
4. **把 F5 只放在逆势吸筹型品种中检验**
5. **最终目标是形成‘风格识别 + 条件化因子 + 低频交易’的框架**

这会比继续做一轮“全市场统一相关性研究”更接近真实可交易结果。
