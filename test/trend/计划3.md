# 计划3：基于结论2的量化信号指标发明与趋势收益检验

> 基于结论2已证明的核心结论：**主力资金连续趋势与价格方向存在69.2%的共振率，5天+趋势更达75.3%，趋势幅度与价格幅度正相关（r=0.246）**，本计划在此基础上发明一系列可操作的量化交易信号（**多空对称**），并检验其持有3/5/10天后的实际趋势收益。

---

## 一、研究框架

### 基本逻辑

结论2已从统计意义上证明了"主力资金趋势 → 价格趋势"的关联性，但停留在段级别的整体统计。计划3要做的是：

**将"主力趋势共振"这一规律，拆解成若干个有明确触发条件的量化交易信号（多空对称），逐一检验每个信号的实际盈利能力。**

信号设计原则：
1. 每个指标背后必须有一个**合理的市场假设**（不是数据挖掘）
2. 每个指标都包含**做多版本（LONG）和做空版本（SHORT）两个对称变体**——既然主力上升能预测价格上涨，按对称性主力下降也应能预测价格下跌
3. 信号触发后，**在信号当天收盘价买入/卖出**（无未来数据偷看）
4. 统一测量**持有3天/5天/10天后的收益率**
5. 计算各持有期的**胜率、平均收益、中位数收益、最大亏损**

### 多空对称性原则

**重要**：结论2的所有发现本质上是方向无关的——主力资金的连续方向变化与价格方向同步。因此每个指标都必须同时实现做多和做空两个版本：

| 维度 | 做多（LONG） | 做空（SHORT） |
|------|-------------|---------------|
| 主力方向 | 连续上升 | 连续下降 |
| 起始位置 | 极端负值/低位 | 极端正值/高位 |
| 散户配合 | 散户同期下降（背离） | 散户同期上升（背离） |
| 收益计算 | `(close[t+n] - close[t]) / close[t]` | `-(close[t+n] - close[t]) / close[t]` |

做空信号的胜率检验同样以"扣除手续费前的方向是否对了"为准，分别独立统计。最终评估时，多空信号的整体胜率应该接近（如果差异巨大，说明市场存在结构性不对称，本身也是有价值的发现）。

### 统一评估框架

每个指标产生的每次买入信号记录为一条"交易记录"，包含：

| 字段 | 说明 |
|------|------|
| `variety_id` | 品种 |
| `signal_date` | 信号触发日期（开仓日） |
| `entry_price` | 信号当天收盘价 |
| `indicator_name` | 所属指标名称 |
| `direction` | `LONG`（做多）或 `SHORT`（做空） |
| `indicator_value` | 指标的量化数值（用于强弱分组） |
| `return_3d` | 持有3天后收益率（做空时取反） |
| `return_5d` | 持有5天后收益率（做空时取反） |
| `return_10d` | 持有10天后收益率（做空时取反） |
| `win_3d` | 3天后方向是否正确（1/0） |
| `win_5d` | 5天后方向是否正确（1/0） |
| `win_10d` | 10天后方向是否正确（1/0） |

**收益率计算公式**：
- LONG： `return_nd = (close[t+n] - close[t]) / close[t]`
- SHORT：`return_nd = -(close[t+n] - close[t]) / close[t] = (close[t] - close[t+n]) / close[t]`

**基准对照**：对所有日期随机买入的平均收益（随机基准），用于判断各指标是否真的有超额预测价值。

## 二、量化指标设计

### 指标1：主力底部/顶部3连变信号（MF_Edge3）

**LONG 版本（MF_Bottom3）**

**假设**：主力资金从阶段低点开始连续3天递增，说明主力开始介入（由空转多），此时跟随做多有概率获利。

**触发条件**：
```
连续3个交易日满足：main_force[t] > main_force[t-1] > main_force[t-2]
```
即 t-2 到 t 的 main_force 严格单调递增，不允许任何一天回调。

**开仓时机**：第3天（确认日）收盘价**买入**。

---

**SHORT 版本（MF_Top3）**

**假设**：主力资金从阶段高点开始连续3天递减，说明主力开始撤离/反手做空（由多转空），此时跟随做空有概率获利。

**触发条件**：
```
连续3个交易日满足：main_force[t] < main_force[t-1] < main_force[t-2]
```
即 t-2 到 t 的 main_force 严格单调递减。

**开仓时机**：第3天（确认日）收盘价**卖出（开空）**。

---

**衍生分组**（检验"边缘启动"是否比中性区启动更有效）：

| 分组 | LONG 定义 | SHORT 定义 |
|------|----------|-----------|
| **极端边缘启动** | main_force[t-2] < 历史 20 百分位 | main_force[t-2] > 历史 80 百分位 |
| **普通边缘启动** | main_force[t-2] 在 20~50 百分位 | main_force[t-2] 在 50~80 百分位 |
| **中性区启动** | main_force[t-2] 在 50 百分位附近 | 同左 |

**核心假设验证**：从极端边缘启动的3连变信号，比中性区启动的3连变信号，未来3/5/10天收益更高。LONG 与 SHORT 的胜率应大致对称。

---

### 指标2：趋势加速度信号（MF_Accel）

**假设**：结论2证明趋势幅度与价格幅度正相关（r=0.246）。那么同样是3连变，加速度越强（每天变化越来越大），是否预示着更大的价格变动？

**定义**：计算3连变期间的加速度：
```
delta1 = main_force[t-1] - main_force[t-2]   # 第1步变化
delta2 = main_force[t]   - main_force[t-1]   # 第2步变化
accel  = |delta2| - |delta1|                  # 加速度（正=加速，负=减速）
```

**触发条件**：在满足3连变的基础上，按加速度分为两组：

**LONG（在3连升基础上）**

| 分组 | 定义 | 举例 |
|------|------|------|
| **加速型** | 第2步涨幅 > 第1步涨幅 | -50→-44→-32（+6，+12，加速） |
| **减速型** | 第2步涨幅 ≤ 第1步涨幅 | -50→-38→-33（+12，+5，减速） |

**SHORT（在3连降基础上）**

| 分组 | 定义 | 举例 |
|------|------|------|
| **加速型** | 第2步跌幅 > 第1步跌幅 | 50→44→32（-6，-12，加速下滑） |
| **减速型** | 第2步跌幅 ≤ 第1步跌幅 | 50→38→33（-12，-5，减速下滑） |

**核心假设验证**：加速型信号（无论多空）的后续价格变动幅度，显著大于减速型。

---

### 指标3：趋势持续天数信号（MF_Duration）

**假设**：结论2明确发现"5天+趋势的方向一致率（75.3%）远高于3天趋势（47.1%）"。因此，等待更长的趋势确认，虽然入场晚，但信号更可靠，最终收益更高。**该规律在做空方向上同样适用**。

**触发条件**：检验3种不同确认天数的入场时机（多空对称）：

| 信号版本 | LONG 触发 | SHORT 触发 | 牺牲 / 收益权衡 |
|---------|----------|-----------|---------------|
| **D3** | 连续3天 main_force 单调上升 | 连续3天 main_force 单调下降 | 早入场，信号噪音多 |
| **D5** | 连续5天 main_force 单调上升 | 连续5天 main_force 单调下降 | 错过前2天行情，信号更可靠 |
| **D7** | 连续7天 main_force 单调上升 | 连续7天 main_force 单调下降 | 错过前4天行情，最可靠但最稀有 |

**核心假设验证**：D5和D7触发后的3/5/10天收益胜率，是否显著高于D3？多空两个方向应表现一致。

---

### 指标4：主散背离共振信号（MS_Divergence）

**假设**：结论2证明散户是"反向指标"（67%概率与价格反向）。当主力与散户方向相反时，背离信号最强：
- **主力上升 + 散户下降** = 主力在吸筹而散户在割肉，**最强做多信号**
- **主力下降 + 散户上升** = 主力在派发而散户在接盘，**最强做空信号**

**LONG 触发**：在主力3连升基础上，要求散户同期下降
```
main_force[t] > main_force[t-1] > main_force[t-2]
且 retail 同期均值为下降趋势
```

**SHORT 触发**：在主力3连降基础上，要求散户同期上升
```
main_force[t] < main_force[t-1] < main_force[t-2]
且 retail 同期均值为上升趋势
```

**背离强度量化**：
```
divergence_score = |main_force_delta_avg - retail_delta_avg|
（主散方向相反时数值最大，越大背离越强）
```

**分组对比**：

| 分组 | LONG 条件 | SHORT 条件 |
|------|----------|-----------|
| **强共振信号** | 主力3连升 + 散户3连降 | 主力3连降 + 散户3连升 |
| **单边主力信号** | 主力3连升 + 散户方向不定 | 主力3连降 + 散户方向不定 |
| **弱共振信号** | 主力3连升 + 散户也在上升 | 主力3连降 + 散户也在下降 |

**核心假设验证**：强共振信号的胜率和收益，是否显著高于单边主力信号？做多和做空两侧的强共振胜率应大致对称。

---

### 指标5：主力趋势总幅度门槛信号（MF_Magnitude）

**假设**：结论2发现幅度与价格变化正相关（r=0.246）。因此设定幅度门槛——只有当主力趋势的总幅度（绝对值）超过历史中位数的N倍时，才触发开仓信号（过滤掉微弱的趋势噪音）。

**幅度定义**：
```
magnitude = |main_force[t] - main_force[t-2]|   # 3连变期间的总幅度（绝对值）
direction = sign(main_force[t] - main_force[t-2])  # 决定多/空方向
```

**触发条件**（在满足3连变基础上，多空对称）：

| 分组 | 幅度门槛 | LONG / SHORT |
|------|---------|--------------|
| **高幅度** | magnitude > 该品种历史 |变化幅度| 的 75 百分位 | 上升→LONG；下降→SHORT |
| **中幅度** | magnitude 在 25~75 百分位之间 | 同上 |
| **低幅度** | magnitude < 该品种历史 |变化幅度| 的 25 百分位 | 同上 |

**核心假设验证**：高幅度触发的信号，后续价格表现是否显著优于低幅度信号？验证"幅度越大，价格跟随越强"，多空两个方向上的效应是否对称。

---

### 指标6：主力区域突破信号（MF_BreakZone）

**假设**：当主力资金不仅连续变化，而且穿越关键的分位区域边界，说明力量格局发生质变（如做空主力平仓离场，或做多主力获利了结），价格突破/下破的可能性更大。

**LONG 触发**：
```
start_zone（t-2时）：极端负值区（< 10百分位）
end_zone（t时）：升入负值区或以上（≥ 10百分位）
且满足3天连续上升
```
即主力资金从极端低谷**升穿** 10 百分位分界线。

**SHORT 触发**：
```
start_zone（t-2时）：极端正值区（> 90百分位）
end_zone（t时）：跌入正值区或以下（≤ 90百分位）
且满足3天连续下降
```
即主力资金从极端高峰**跌穿** 90 百分位分界线。

**对照组**：同样是3连变，但完全在一个区域内移动（未跨越任何关键百分位边界）。

**核心假设验证**：跨越关键边界的趋势，比在同一区域内震荡的趋势，后续价格收益更高。多空两个方向上的"突破效应"应该对称存在。

---

### 指标7：综合评分信号（Composite_Score）

**假设**：以上各指标都从不同维度刻画"主力趋势质量"。将多个维度加权合成一个综合评分，高分信号应该比任何单一指标的信号都更可靠。**评分对多空对称**——只要主力趋势方向明确，即可分别打分。

**评分构成**（多空适用同一套规则）：

| 维度 | 评分规则 | 权重 |
|------|---------|------|
| 趋势持续天数 | 3天=1, 5天=2, 7天+=3 | 30% |
| 加速度 | 加速 = 1分，减速 = 0分 | 20% |
| 边缘区启动 | 极端边缘（前/后10%）=2, 边缘（前/后20%）=1, 其他=0 | 20% |
| 主散背离 | 散户同期反向 = 1，否则 0 | 20% |
| 幅度强度 | 高幅度=2, 中=1, 低=0 | 10% |

**综合评分 = 加权求和（满分约为2.5）**

**信号方向**：根据主力趋势方向决定 LONG / SHORT
- 主力上升趋势 → LONG 信号 + 评分
- 主力下降趋势 → SHORT 信号 + 评分

**触发条件**：综合评分 ≥ 阈值（分别检验 ≥1.0, ≥1.5, ≥2.0 三档），多空独立统计。

**核心假设验证**：评分越高的信号，3/5/10天收益越好，说明多维度过滤是有效的；多空两侧的高分信号胜率应大致对称。

## 三、实验设计

### 3.1 数据准备

与计划2使用相同的数据基础：
- `fut_strength`（main_force, retail）× `fut_daily_close`（收盘价）
- JOIN 后得到约 1,595 条完整记录（55 品种 × ~29 天）
- 需要为每条信号计算 t+3、t+5、t+10 的收盘价，因此需过滤掉尾部没有足够未来数据的信号

### 3.2 信号提取流程

```
对每个品种：
  1. 按日期排序
  2. 遍历所有可能的信号触发点（最早从第3天开始）
  3. 对每个指标，判断是否满足触发条件
  4. 若满足，计算 return_3d / return_5d / return_10d
  5. 记录到统一的信号表
```

注意：**一个交易日可能同时触发多个指标的信号**，各指标独立统计，不合并。

### 3.3 随机基准的构建

从所有有完整未来数据的日期中，随机抽取与各指标信号数相同数量的"随机买入"，重复1000次，计算平均收益分布作为基准。

### 3.4 统计检验

对每个指标的收益序列做：
- **t检验**：均值是否显著大于0（买入后有正收益）
- **与随机基准的z检验**：是否显著优于随机买入
- **分组间Mann-Whitney U检验**：加速型 vs 减速型等分组比较

## 四、核心评估指标

对每个信号指标的 **LONG 和 SHORT 版本分别**输出以下统计表：

| 评估维度 | 持有3天 | 持有5天 | 持有10天 |
|---------|---------|---------|---------|
| 触发次数 | N | N | N |
| 胜率（方向正确比例） | % | % | % |
| 平均收益率 | % | % | % |
| 中位数收益率 | % | % | % |
| 盈亏比 | - | - | - |
| 最大单次亏损 | % | % | % |
| vs 随机基准超额 | % | % | % |
| p值（显著性） | - | - | - |

### 多空对称性检验

为每个指标额外输出一张多空对比表，检验市场是否存在结构性不对称：

| 指标 | LONG 5d 胜率 | SHORT 5d 胜率 | 胜率差 | 是否对称 |
|------|-------------|--------------|--------|---------|
| MF_Edge3 | % | % | % | 是/否 |
| ... | | | | |

如果多空胜率差距 > 10 个百分点，说明该规律存在方向偏见（可能受样本期内的整体趋势偏向影响），需要在结论中明确指出。

**最终排名**：按"5天胜率 × 5天平均收益"的综合得分，对所有指标×方向（LONG/SHORT）的组合分别排名，确定最优信号。

| 板块 | 预期最优指标 | 理由 |

## 七、预期结论方向与局限性说明

### 预期结论

1. **MF_Duration（D5/D7）** 的胜率应高于 D3，但触发频率大幅降低，验证结论2"5天是关键分水岭"
2. **MF_Accel（加速型）** 的后续收益应高于减速型，验证"加速度效应"假设
3. **MS_Divergence（强共振）** 应是胜率最高的单一指标，因为融合了主力和散户两个维度的信息
4. **Composite_Score（高评分档）** 应是综合表现最好的指标，但触发次数最少
5. **多空对称性**：LONG 与 SHORT 信号的胜率差应在 10 个百分点以内。若差距过大，需归因于：
   - 样本期内整体趋势偏向（28天数据可能整体上涨或下跌）
   - 主力资金在多空两侧的行为本质不同（如做空力量更隐蔽）
   - 品种结构性差异（农产品 vs 工业品）

### 局限性

1. **样本量约束**：每品种约29天数据，10天持有期后剩余数据极为有限，10天收益样本最少
2. **过拟合风险**：指标参数是基于结论2的发现设计的，在同一段数据上验证存在循环论证风险——**结论仅在数据积累到120天后才能用于真实策略决策**
3. **未考虑交易成本**：所有收益率均为净价格收益，未扣除手续费和滑点
4. **信号重叠**：同一天可能触发多个指标，跨指标的相关性分析未纳入本计划
