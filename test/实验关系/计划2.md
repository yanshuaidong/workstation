# 计划2：基于主力资金趋势的价格共振研究

> 基于计划1的研究结论和人工观察，提出新假设：**单个 main_force 值无预测价值，但连续 3 个以上值构成的趋势方向与期货价格趋势存在明显共振关系。**

---

## 核心假设

### 假设 H1：趋势共振
当 `main_force` 连续 3 天及以上保持同一方向变化（如从大负值→小负值→正值），期货价格大概率跟随相同方向运动。散户（`retail`）方向相反。

### 假设 H2：底部反转信号
当 `main_force` 处于极端负值区域后开始连续回升（负值绝对值减小），即使尚未转正，期货价格在未来 5 个交易日内**创新低的概率显著低于基准**（随机概率）。市场大概率进入震荡或跟随主力趋势上行。

### 假设 H3：对称性——顶部反转信号
当 `main_force` 处于极端正值区域后开始连续回落（正值减小→负值），价格在未来 5 个交易日内创新高的概率显著降低。

---

## 与计划1的关键区别

| 维度 | 计划1（已完成） | 计划2（本次） |
|------|----------------|--------------|
| **分析单元** | 单日值 | 连续 N 日构成的趋势段 |
| **核心方法** | Pearson/Spearman 相关系数 | 趋势段匹配 + 条件概率统计 |
| **信号定义** | main_force 当日值的大小 | main_force 连续变化方向 + 起止位置 |
| **预测目标** | 次日收益率 | 未来 N 日是否创新高/新低 |
| **最小窗口** | 1 天 | 3 天（趋势最小单元） |

---

## 研究阶段

### 阶段一：趋势识别与标注

**目标**：在 main_force 时间序列上识别并标注所有趋势段。

#### 1.1 趋势定义

一个**上升趋势段**满足：
- 连续 ≥ 3 个交易日，`main_force` 的日环比变化 (`delta`) 累积为正
- 即 `main_force[t] > main_force[t-1]`（允许中间有 1 天小幅回调，容忍度参数化）

一个**下降趋势段**定义对称。

#### 1.2 趋势段属性

每个识别出的趋势段记录以下属性：

| 属性 | 说明 |
|------|------|
| `variety_id` | 品种 |
| `trend_start` | 趋势起始日期 |
| `trend_end` | 趋势结束日期 |
| `duration` | 持续天数 |
| `start_value` | 起始 main_force 值 |
| `end_value` | 结束 main_force 值 |
| `direction` | `up` 或 `down` |
| `start_zone` | 起始位置所在区域：极端负值 / 负值 / 中性 / 正值 / 极端正值（按品种历史百分位划分） |
| `magnitude` | 趋势幅度 = `end_value - start_value` |

#### 1.3 区域划分标准

按每个品种自身 main_force 的分布，划分 5 个区域：
- **极端负值**：< 10% 百分位
- **负值区**：10%~30% 百分位
- **中性区**：30%~70% 百分位
- **正值区**：70%~90% 百分位
- **极端正值**：> 90% 百分位

### 阶段二：趋势共振分析（验证 H1）

**目标**：量化 main_force 趋势段与同期价格趋势的一致性。

#### 2.1 价格趋势识别

对收盘价序列用同样方法识别趋势段。

#### 2.2 共振度计算

对每个 main_force 趋势段，计算其覆盖时间窗口内价格的变化方向：

| 指标 | 公式 | 含义 |
|------|------|------|
| **方向一致率** | `主力上升段中价格也上涨的比例` | 趋势方向是否同步 |
| **趋势重叠度** | `主力趋势段与价格趋势段的时间重叠天数 / 主力趋势段天数` | 两者在时间上是否同步启动 |
| **幅度相关性** | `Corr(主力趋势幅度, 价格变化幅度)` | 主力趋势越强，价格变化越大吗 |

#### 2.3 分组对比

- 按趋势持续天数分组（3天 / 4天 / 5天+ ）：持续越久共振越强吗？
- 按起始区域分组：从极端负值启动的上升趋势，vs 从中性区启动的，共振度有差异吗？
- 按品种板块分组：哪些板块的趋势共振最明显？

#### 2.4 对照组

用 retail 做同样分析，验证散户趋势是否与价格**反向**共振。

---

### 阶段三：条件概率分析（验证 H2 & H3）

**目标**：严格检验"主力趋势回升时，价格不容易创新低"这一假设。

#### 3.1 定义事件

| 事件 | 定义 |
|------|------|
| **A：主力底部回升** | main_force 从极端负值/负值区开始，连续 ≥ 3 天上升 |
| **B：未来 5 日创新低** | `min(close[t+1:t+5]) < min(close[t-4:t])` — 未来 5 天的最低收盘价 < 过去 5 天的最低收盘价 |
| **C：主力顶部回落** | main_force 从极端正值/正值区开始，连续 ≥ 3 天下降 |
| **D：未来 5 日创新高** | `max(close[t+1:t+5]) > max(close[t-4:t])` |

#### 3.2 计算条件概率

| 检验 | 公式 | 期望结果 |
|------|------|---------|
| P(B\|A) | 主力底部回升时创新低的概率 | **显著低于** P(B)（无条件概率） |
| P(D\|C) | 主力顶部回落时创新高的概率 | **显著低于** P(D) |
| P(¬B\|A) | 主力底部回升时**不创新低**的概率 | **显著高于** P(¬B) |

#### 3.3 统计检验

- 对每个品种做 Fisher 精确检验或卡方检验，比较条件概率与基准概率的差异
- 汇总所有品种，做整体的 meta 分析
- 报告：效果显著的品种列表、平均效应量、置信区间

#### 3.4 敏感性分析

调整参数重新检验，确认结论的稳健性：
- 趋势最小天数：3 / 4 / 5
- 未来窗口：3 / 5 / 7 天
- 新低/新高的参照窗口：3 / 5 / 10 天
- 极端区域阈值：10% / 20% / 25% 百分位

### 阶段四：趋势跟踪策略设计与回测

**目标**：将趋势共振关系转化为可操作的交易策略并回测。

#### 4.1 策略定义

**策略 A — 趋势跟随信号**

| 条件 | 动作 |
|------|------|
| main_force 连续 ≥ 3 天上升，且起始于负值区 | 做多信号（买入） |
| main_force 连续 ≥ 3 天下降，且起始于正值区 | 做空信号（卖出） |
| main_force 趋势中断（反转 1 天以上） | 平仓 |

**策略 B — 新低保护信号**

| 条件 | 动作 |
|------|------|
| 触发做多信号（同策略A）后 | 持仓，止损设在过去 5 日最低价 |
| 若 5 天内未创新低 | 持有或移动止损 |
| 主力趋势反转 | 平仓 |

**策略 C — 主散共振策略**

| 条件 | 动作 |
|------|------|
| main_force 上升趋势 + retail 下降趋势 同时出现 | 强做多信号 |
| main_force 下降趋势 + retail 上升趋势 同时出现 | 强做空信号 |

#### 4.2 回测指标

| 指标 | 说明 |
|------|------|
| 趋势段胜率 | 信号方向与实际价格方向一致的趋势段比例 |
| 平均趋势段收益 | 每段持仓的平均收益率 |
| 信号触发频率 | 平均多少天触发一次信号 |
| 最大连续亏损段 | 连续方向错误的段数 |
| 未来 5 日新低概率 | 做多信号触发后未来 5 日实际创新低的比例 |

#### 4.3 与计划1策略对比

将策略 A/B/C 的表现与计划1中的"综合信号策略"做横向对比，重点关注：
- 胜率是否提升
- 假信号是否减少
- 在计划1中表现差的品种（鸡蛋、甲醇等），趋势方法是否改善

## 数据局限与注意事项

1. **样本量仍然偏小**：28 个交易日内能识别出的完整趋势段有限（预计每品种 3~6 段），条件概率的置信度取决于趋势段总量
2. **趋势识别的参数敏感性**：定义"连续上升"时是否允许中间回调 1 天，会显著影响识别出的趋势段数量，需通过敏感性分析确定最优参数
3. **前瞻偏差**：趋势段的识别需要回顾至少 3 天的数据，实际交易中只有在第 3 天结束时才能确认趋势成立，需在回测中正确处理时间对齐
4. **与计划1结论的衔接**：重点关注计划1中已证明有效的品种（有色金属板块），看趋势方法能否进一步提升信号质量
