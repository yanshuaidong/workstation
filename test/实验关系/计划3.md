# 计划3：资金曲线微分特征与价格趋势关系研究

> 基于计划1（单日值弱相关）和计划2（趋势段强共振 69.2%）的递进结论，本次从**微积分视角**切入：将 main_force / retail 的时间序列视为离散曲线，提取其**一阶导（斜率/速度）**和**二阶导（加速度/曲率变化）**，研究这些微分特征与价格趋势之间的量化关系。

---

## 核心思路

计划2已证明"连续趋势方向"包含强信号（方向一致率 69.2%）。但趋势段分析只提取了**方向**这一个维度，忽略了更丰富的动态信息——趋势在加速还是减速？拐点何时出现？本计划通过微分分析，将资金曲线的"形状信息"量化为可计算的特征。

### 直觉类比

把 main_force 的每日值想象成散点图上的点，虽然是离散的，但相邻点之间天然存在斜率：

- **一阶导（斜率）**：资金力量的变化速度——主力在加仓还是减仓？多快？
- **二阶导（加速度）**：资金力量的变化加速度——加仓在提速还是在放缓？趋势是否即将反转？

---

## 阶段一：离散微分特征计算

### 1.1 一阶导数（斜率）定义

对每个品种的 main_force 和 retail 时间序列，分别计算两种一阶导：

| 特征名 | 公式 | 含义 |
|--------|------|------|
| **相邻斜率** `slope_1` | `(f[t] - f[t-1]) / 1` | 相邻两点的瞬时变化率，最细粒度的速度 |
| **跳点斜率** `slope_2` | `(f[t+1] - f[t-1]) / 2` | 跳过当前点，用前后两点构成的斜率（类似中心差分），可平滑单日噪声 |

其中 `f[t]` 为第 t 天的 main_force（或 retail）值。

**为什么需要两种斜率**：
- `slope_1` 灵敏度高，但容易受单日异常值干扰
- `slope_2` 相当于2日窗口的中心差分，本质上是一种去噪——当中间一个点是异常值时，`slope_2` 仍能反映真实趋势方向

### 1.2 二阶导数（加速度）定义

| 特征名 | 公式 | 含义 |
|--------|------|------|
| **加速度** `accel` | `slope_1[t] - slope_1[t-1]` | 即 `f[t] - 2*f[t-1] + f[t-2]`，资金变化的加速度 |
| **平滑加速度** `accel_smooth` | `slope_2[t] - slope_2[t-1]` | 基于跳点斜率的二阶差分，更稳定 |

### 1.3 派生特征

| 特征名 | 公式 | 含义 |
|--------|------|------|
| `slope_sign` | `sign(slope_1)` | 斜率方向：+1（上升）/ -1（下降）/ 0（持平） |
| `accel_sign` | `sign(accel)` | 加速度方向：+1（加速）/ -1（减速）/ 0（匀速） |
| `slope_accel_combo` | `(slope_sign, accel_sign)` 的组合 | 4 种状态（见下表） |
| `slope_ma3` | `slope_1` 的 3 日均值 | 平滑后的趋势速度 |
| `slope_ma5` | `slope_1` 的 5 日均值 | 更稳定的趋势速度 |

### 1.4 四象限状态分类

将斜率方向和加速度方向组合，形成 4 种"资金运动状态"：

| 状态 | 斜率 | 加速度 | 物理含义 | 市场含义 |
|------|------|--------|----------|----------|
| **加速上升** | + | + | 向上且加速 | 主力加仓力度越来越大，强势做多 |
| **减速上升** | + | - | 向上但减速 | 主力仍在加仓但力度减弱，可能接近拐点 |
| **加速下降** | - | - | 向下且加速 | 主力减仓/做空力度加大，强势看空 |
| **减速下降** | - | + | 向下但减速 | 主力做空力度减弱，可能即将企稳反弹 |

> 对 retail 做同样分类，但市场含义相反（散户加速做多可能是反向信号）。

## 阶段二：微分特征与价格关联分析

### 2.1 基础相关性

对每个品种，计算以下相关系数矩阵：

| 自变量 | 因变量 | 分析目的 |
|--------|--------|----------|
| `main_slope_1[t]` | `price_return[t]` | 主力斜率与当日收益率的同期关系 |
| `main_slope_1[t]` | `price_return[t+1]` | 主力斜率对次日价格的领先性 |
| `main_slope_2[t]` | `price_return[t+1]` | 跳点斜率的领先性（预期更稳定） |
| `main_accel[t]` | `price_return[t+1]` | 加速度对次日价格的预测力 |
| `retail_slope_1[t]` | `price_return[t+1]` | 散户斜率对价格的预测力 |
| `main_slope_1[t] - retail_slope_1[t]` | `price_return[t+1]` | 主散斜率差对价格的预测力 |

与计划1对比：计划1用 main_force 原始值做相关（r=0.108），**这里期望斜率和加速度能捕捉到更强的信号**，因为计划2已证明"变化方向"比"绝对值"包含更多信息。

### 2.2 四象限条件收益率分析

这是本计划最核心的分析：

**在 main_force 处于不同运动状态时，次日/未来 N 日的价格收益率是否有显著差异？**

| 分组 | 计算内容 |
|------|---------|
| 加速上升期间 | 平均次日收益率、平均未来3日累计收益率、正收益概率 |
| 减速上升期间 | 同上 |
| 加速下降期间 | 同上 |
| 减速下降期间 | 同上 |

**统计检验**：
- 对四组收益率做 Kruskal-Wallis 检验（非参数方差分析），判断四种状态下的收益率分布是否有显著差异
- 两两对比：加速上升 vs 加速下降（预期差异最大）、减速上升 vs 加速上升（拐点信号检验）

### 2.3 拐点检测与价格反转

**核心假设**：二阶导符号发生变化（加速度从正变负或从负变正）的时刻，即为资金曲线的拐点，可能预示价格趋势即将改变。

具体检测：

| 拐点类型 | 条件 | 预期价格反应 |
|---------|------|-------------|
| **上升拐点**（由加速上升→减速上升） | `accel[t-1] > 0` 且 `accel[t] < 0`，同时 `slope[t] > 0` | 上涨趋势可能见顶，未来 N 日涨幅收窄或转跌 |
| **下降拐点**（由加速下降→减速下降） | `accel[t-1] < 0` 且 `accel[t] > 0`，同时 `slope[t] < 0` | 下跌趋势可能见底，未来 N 日跌幅收窄或转涨 |
| **趋势启动点**（由减速下降→加速上升） | `slope[t-1] < 0, accel[t-1] > 0` → `slope[t] > 0, accel[t] > 0` | 空翻多拐点，强买入信号 |
| **趋势终结点**（由减速上升→加速下降） | `slope[t-1] > 0, accel[t-1] < 0` → `slope[t] < 0, accel[t] < 0` | 多翻空拐点，强卖出信号 |

对每种拐点，统计其出现后未来 1/3/5 日的平均价格变化，与无条件基准对比。

## 阶段三：主力与散户微分特征的对比分析

### 3.1 主散斜率背离

| 指标 | 定义 | 含义 |
|------|------|------|
| **斜率背离度** | `main_slope_1 - retail_slope_1` | 主力和散户变化方向的分歧程度 |
| **斜率背离符号** | `sign(main_slope) ≠ sign(retail_slope)` | 是否处于背离状态 |
| **背离强度** | `abs(main_slope) + abs(retail_slope)` 当方向相反时 | 双方力度之和（方向相反时） |

**研究问题**：
- 主散斜率同时为正（双方都在加速做多）vs 主散斜率背离（主力加速做多+散户加速做空）时，价格走势有何差异？
- 背离持续天数与后续价格变动的关系

### 3.2 主散加速度特征对比

| 场景 | 条件 | 预期 |
|------|------|------|
| 主力加速+散户减速 | `main_accel > 0, retail_accel < 0` | 强趋势信号（主力主导） |
| 主力减速+散户加速 | `main_accel < 0, retail_accel > 0` | 趋势可能反转（散户接盘） |
| 双方同时加速 | `main_accel > 0, retail_accel > 0` | 市场分歧，方向不明确 |

## 阶段四：微分信号策略设计与回测

### 4.1 策略 D — 加速度信号策略

| 条件 | 动作 |
|------|------|
| main_force 处于"加速上升"状态连续 ≥ 2 天 | 做多 |
| main_force 处于"加速下降"状态连续 ≥ 2 天 | 做空 |
| 状态进入"减速"区间 | 平仓（不反手） |

### 4.2 策略 E — 拐点反转策略

| 条件 | 动作 |
|------|------|
| 检测到"趋势启动点"（空翻多拐点） | 做多 |
| 检测到"趋势终结点"（多翻空拐点） | 做空 |
| 持仓 N 天后自动平仓 | 固定持仓周期（参数化：3/5 天） |

### 4.3 策略 F — 主散微分背离策略

| 条件 | 动作 |
|------|------|
| 主力斜率 > 0 且散户斜率 < 0 且持续 ≥ 2 天 | 做多（跟主力，逆散户） |
| 主力斜率 < 0 且散户斜率 > 0 且持续 ≥ 2 天 | 做空 |
| 背离消失（同向） | 平仓 |

### 4.4 策略 G — 综合微分信号策略

将斜率、加速度、主散背离三个维度融合为一个综合评分：

```
score = w1 * normalize(main_slope_ma3)
      + w2 * normalize(main_accel)
      + w3 * normalize(main_slope - retail_slope)
```

权重 `w1, w2, w3` 先等权（各1/3），后续可根据阶段二的相关性强度调整。

| 条件 | 动作 |
|------|------|
| `score > +threshold` | 做多 |
| `score < -threshold` | 做空 |
| `-threshold ≤ score ≤ +threshold` | 观望/平仓 |

### 4.5 回测指标

| 指标 | 说明 |
|------|------|
| 各策略胜率 | 与计划1（51.7%）、计划2（52.0%）对比 |
| 各策略盈利品种比例 | 与计划1（56%）、计划2（64%）对比 |
| 各策略平均收益 | 与前两轮对比 |
| 信号触发频率 | 加速度信号 vs 趋势信号 vs 单日值信号的触发密度 |
| 拐点信号的预测准确率 | 拐点出现后价格确实反转的比例 |
| 分板块表现 | 重点关注黑色系、能源化工（计划2最优板块）和有色金属（计划1最优板块） |

### 4.6 与计划1/计划2策略的系统对比

| 维度 | 计划1 综合信号 | 计划2 策略A | 计划3 策略D~G |
|------|--------------|-----------|-------------|
| 信号来源 | 原始值 | 趋势方向 | 斜率 + 加速度 |
| 信号粒度 | 每日 | 趋势段（3~N 天） | 每日（但含多日信息） |
| 反应速度 | 最快（无延迟） | 最慢（需确认趋势） | 中等（1~2日延迟） |
| 抗噪声能力 | 弱 | 强 | 中等（跳点斜率较强） |

## 数据局限与注意事项

1. **样本量约束**：28 个交易日可计算约 26 个一阶导值、25 个二阶导值，每品种的拐点事件数预计仅 3~8 个，统计检验效力有限
2. **噪声放大效应**：微分天然对噪声敏感——原始信号的小扰动在一阶导中被放大，在二阶导中进一步放大。因此 `slope_2`（跳点斜率）和 `slope_ma3/ma5`（移动均值平滑）是必要的去噪手段
3. **跳点斜率的物理含义**：`slope_2 = (f[t+1] - f[t-1]) / 2` 使用了 t+1 的数据，在实盘中存在**前瞻偏差**——回测时需注意，该特征用于**分析研究**而非交易信号；交易信号应仅使用 `slope_1` 和基于历史数据的 `slope_ma`
4. **与计划2的衔接**：计划2已证明 5 天+ 趋势具有强信号意义，本计划的加速度分析可以解释"为什么5天是分水岭"——可能是因为持续5天以上的趋势通常经历了"加速→匀速"阶段，信号已经充分确认
