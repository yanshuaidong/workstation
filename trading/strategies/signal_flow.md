# 策略信号流程图

本文档说明当前策略中四类信号的生成和执行流程：

- 开多信号
- 开空信号
- 平多信号
- 平空信号

## 一、理论信号生成总流程

```mermaid
flowchart TD
    A[读取某个品种的历史数据<br/>主力强度、散户强度、收盘价] --> B[判断数据连续性<br/>最近七条是否连续<br/>最近三条是否连续]
    B --> C[计算背景窗口<br/>取当前日前第六条到第二条<br/>共五个主力强度值]
    C --> D[计算触发窗口<br/>最近两次主力强度变化<br/>最近两次散户强度变化]
    D --> E{判断是否触发原始信号}

    E --> F{是否满足开多条件}
    F -->|是| G[生成理论开多信号]
    F -->|否| H{是否满足开空条件}

    H -->|是| I[生成理论开空信号]
    H -->|否| J{是否满足平多条件}

    J -->|是| K[生成原始平多信号]
    J -->|否| L{是否满足平空条件}

    L -->|是| M[生成原始平空信号]
    L -->|否| N[当日无信号]

    G --> O[进入理论状态机]
    I --> O
    K --> O
    M --> O

    O --> P{当前理论状态是什么}

    P -->|当前无持仓状态<br/>且出现开多| Q[保留开多信号<br/>理论状态变为持有多头]
    P -->|当前无持仓状态<br/>且出现开空| R[保留开空信号<br/>理论状态变为持有空头]
    P -->|当前持有多头<br/>且出现平多| S[保留平多信号<br/>理论状态变为无持仓]
    P -->|当前持有空头<br/>且出现平空| T[保留平空信号<br/>理论状态变为无持仓]
    P -->|状态不匹配| U[丢弃该平仓信号或忽略重复开仓]

    Q --> V[写入理论信号表]
    R --> V
    S --> V
    T --> V
```

## 二、四类信号判断条件

```mermaid
flowchart TD
    A[某交易日] --> B{开多条件}
    B -->|最近七条数据连续<br/>背景五个主力强度都小于零<br/>第五个背景值是五个里最低<br/>主力强度连续两次上升<br/>散户强度连续两次下降| C[开多信号]

    A --> D{开空条件}
    D -->|最近七条数据连续<br/>背景五个主力强度都大于零<br/>第五个背景值是五个里最高<br/>主力强度连续两次下降<br/>散户强度连续两次上升| E[开空信号]

    A --> F{平多条件}
    F -->|最近三条数据连续<br/>当前主力强度减去前第二条主力强度小于零<br/>并且理论状态当前持有多头| G[平多信号]

    A --> H{平空条件}
    H -->|最近三条数据连续<br/>当前主力强度减去前第二条主力强度大于零<br/>并且理论状态当前持有空头| I[平空信号]
```

## 三、真实交易执行流程

理论信号不等于真实交易。系统每天先计算全品种理论信号，然后先处理平仓，再处理开仓。

```mermaid
flowchart TD
    A[每日运行开始] --> B[计算全部品种理论信号]
    B --> C[先处理平仓信号]
    C --> D{理论平仓信号<br/>是否匹配真实持仓}
    D -->|品种一致<br/>方向一致<br/>理论周期一致| E[按当日收盘价真实平仓]
    D -->|不匹配| F[不执行平仓]

    E --> G[记录今日已平仓品种]
    F --> G

    G --> H[再处理开仓建议]
    H --> I{开仓信号是否通过筛选}
    I -->|在池子中<br/>未持仓<br/>今日未刚平仓<br/>仓位未满<br/>板块不冲突| J[按当日收盘价真实开仓]
    I -->|不满足任一条件| K[只记录拒绝原因<br/>不真实开仓]

    J --> L[更新资金曲线]
    K --> L
```

## 四、关键规则说明

### 开多信号

开多信号要求最近七条数据连续，背景窗口里的五个主力强度值全部小于零，并且第五个背景值是五个值中最低的一个。同时，最近两次主力强度变化都要向上，最近两次散户强度变化都要向下。

这表示：主力强度先处在负值区域并形成低点，然后主力开始连续回升，而散户连续走弱。

### 开空信号

开空信号要求最近七条数据连续，背景窗口里的五个主力强度值全部大于零，并且第五个背景值是五个值中最高的一个。同时，最近两次主力强度变化都要向下，最近两次散户强度变化都要向上。

这表示：主力强度先处在正值区域并形成高点，然后主力开始连续回落，而散户连续走强。

### 平多信号

平多信号要求最近三条数据连续，并且当前主力强度减去前第二条主力强度小于零。

不过，原始平多条件成立后，还必须经过理论状态机过滤：只有当前理论状态已经持有多头时，平多信号才会被保留。

### 平空信号

平空信号要求最近三条数据连续，并且当前主力强度减去前第二条主力强度大于零。

不过，原始平空条件成立后，也必须经过理论状态机过滤：只有当前理论状态已经持有空头时，平空信号才会被保留。

## 五、一句话总结

系统先用主力强度和散户强度生成理论上的开多、开空、平多、平空信号；再用理论状态机过滤不合理的平仓；最后真实交易只执行和当前真实持仓、仓位限制、池子配置都匹配的信号。
